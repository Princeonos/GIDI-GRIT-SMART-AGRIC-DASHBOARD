# GIDI GRIT SMART AGRIC DASHBOARD

A smart agriculture PWA designed for real-time soil moisture monitoring and automated irrigation control. This project serves as the technical foundation for my research at Nottingham Trent University.

## 🚀 Key Features
* **Real-time Synchronization:** Powered by Firebase NoSQL database for instant data updates.
* **Interactive Dashboard:** Visual analytics using React and Recharts to track soil health.
* **Automated Logic:** Smart irrigation control based on sensor thresholds.
* **Modern Stack:** Built with Vite, Tailwind CSS, and Lucide icons.

## 🛠️ Tech Stack
* **Frontend:** React.js, Vite
* **Styling:** Tailwind CSS
* **Backend/Database:** Firebase Realtime Database
